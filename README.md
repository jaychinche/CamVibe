#  CamVibe
This full-stack video calling app replicates Zoom’s key features, such as peer-to-peer video calls, real-time messaging, and screen sharing.

# Project Overview:

# Technologies Used:
* Backend: MongoDB, Bcrypt, Crypto, Socket.io, Express.js
* Frontend: HTML, CSS, Material UI, MTC, React, Axios
* Libraries/Tools: CORS, HTTP-Status, Mongoose
  
# Key Features:
* Video Calls
* Real-Time Messaging (chat)
* Screen Sharing
* Secure Authentication
